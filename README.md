# Weesly
Медиа-сервис для предпринимателей

## Доступные скрипты

Для запуска проекта локально:

### `npm install`

Устанавливает зависимости

### `npm start`

Запускает приложение\
Откройте [http://localhost:3000](http://localhost:3000) в браузере.


### `npm run build`

Создает билд в папку `build`.\
